# AI链学习室

基于Spring Boot的在线考试与视频学习系统，包含AI智能组卷、自动批阅、视频管理等功能。

## 功能特性

- **考试系统**：支持AI智能组卷、在线考试、自动批阅、考试记录管理
- **题库管理**：题目增删改查、分类管理、Excel导入、AI生成题目
- **视频学习**：视频上传管理、分类管理、观看统计、点赞收藏
- **内容管理**：轮播图管理、系统公告发布、数据统计仪表盘
- **文件存储**：集成MinIO实现文件上传下载

## 技术栈

- Spring Boot + MyBatis Plus
- Knife4j API文档
- MinIO文件存储
- MySQL数据库
- Redis缓存
- Lombok & Swagger2

## 模块划分

- 控制层：RESTful API设计，包含权限验证
- 服务层：基于MyBatis Plus的CRUD操作
- 数据模型：包含考试、试卷、题目、视频等核心实体
- 工具类：Excel处理、IP工具、响应数据封装

## 接口文档

系统提供完整的API文档，包含：
- 用户认证接口
- 考试管理接口
- 题库管理接口
- 视频管理接口
- 数据统计接口
- 系统设置接口

## 安装部署

1. 克隆项目：`git clone https://gitee.com/AimerUna/ai-chain-learning-room.git`
2. 创建MySQL数据库并导入表结构
3. 配置application.yml中的数据库连接和MinIO参数
4. 执行`mvn spring-boot:run`启动项目

## 使用说明

系统包含两个管理端：
- 考试管理端：处理试卷生成、考试记录、题库维护
- 视频管理端：管理视频内容、分类、审核
